♻️ Smart Recycling & Reward System

📌 Overview

The Smart Recycling & Reward System is a web-based platform designed to promote responsible waste management by encouraging users to recycle through a reward-based mechanism. The system allows users to submit recyclable waste using Smart Bin codes, request pickup services, and earn reward points.

🌐 Live Demo

👉 https://Smart-recycling-reward-system.vercel.app⁠


🚀 Features


🔐 User Authentication (JWT-based login/signup)

♻️ Smart Bin Code Submission

🚚 Pickup Request System

🎁 Reward Points System

📊 User Dashboard (history & rewards)

🛠️ Admin Dashboard (verification & management)

📦 Secure Database Management

📱 Responsive UI


🏗️ Tech Stack


Frontend: React.js

Backend: Node.js, Express.js

Database: MongoDB

Authentication: JWT
